#
# each embedded code follows this rule:
# name | options  | link or embedded code
#
#
# options can be:
#  for Puppybrowser: -w= -h= -x= -y=
#
# if the options contain -run=
#  then Puppybrowser will not be used, but another program
#
# if option contains -converter=
#  then the link will be converted
#
# currently this converter is available:
# -converter=youtube2swf
# here the link can be a youtube page, the converter tries to find and play the main Flashvideo in it.
#
#

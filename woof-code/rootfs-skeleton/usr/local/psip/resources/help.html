How do I set PSIP up?

1 DOWNLOAD PSIP 0.9.8 Beta

2 Register your SIP hardware or Softphone

3 Click on configure / Edit Account

from main menu and add Gizmo account / SIP details

Example?


Change 'Your Sip URL:'
eg. sip:charnisingh@proxy01.sipphone.com

and 'Username' to for example
charnisingh

add your gzmo password
other settings stay the same

then click on [OK]

OK configured, now what?


The latest PSIP has four buttons

CALL (Green button) - make a call, select who you wish to call and click OK,
the person will contacted or their voice mail activated if not available
Press Hangup to finish the call
ANSWER (Orange button) - If you hear someone calling you press answer to connect and hangup when finished
HANGUP (Red Button) - Ends call
CHAT - Text chat if audio is not clear

How do you know who is online with PSIP?


Select Buddy List from the configure pull down menu
